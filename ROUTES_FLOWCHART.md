流程图\n\n```\n┌─────────────────────┐\n│   GET /v1/          │\n└──────────┬──────────┘\n           │\n           ▼\n┌─────────────────────────┐\n│  返回成功响应            │\n│  {\"message\":\"Hello 2. 注册验证路由\n\n### 2.1 检查手机号是否存在\n\n#### 路由信息\n- **路径**: `POST /v1/auth/signup/phone/exist`\n- **处理器**: `SignupController.IsPhoneExist`\n- **请求体**: `SignupPhoneExistRequest`\n- **验证**: `ValidateSignupPhoneExist`\n\n#### 流程图\n\n```\n┌──────────────────────────────┐\n│ POST /v1/auth/signup/         │\n│ phone/exist                   │\n└──────────┬─────────────────────┘\n 执行请求验证                  │\n│ ValidateSignupPhoneExist      │\n└──────────┬─────────────────────┘\n           │\n      ┌────┴────┐\n      │ 验证失败  │ (中止请求)\n      ├─────────┘\n      │\n      ▼\n┌──────────────────────────────┐\n│ 查询用户模型                  │\n│ user.IsPhoneExist(phone)      │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 返回 JSON 响应                │\n│ {\"exist\": bool}              │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ HTTP 200 OK                  │\n└──────────────────────────────┘\n```\n\n#### 实现代码\n```go\nfunc (c *SignupController) IsPhoneExist(ctx *gin.Context) {\n    req := requests.SignupPhoneExistRequest{}\n    \n    if ok := Redis、base64Captcha\n\n#### 流程图\n\n```\n┌──────────────────────────────┐\n│ POST /v1/auth/verify-codes/   │\n│ captcha                       │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ VerifyCodeController         │\n│ .ShowCaptcha()               │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 调用验证码包                  │\n│ captcha.NewCaptcha()         │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 生成验证码                    │\n│ .GenerateCaptcha()            │\n│ • 生成唯一 ID                 │\n│ • 生成 Base64 编码的图片       │\n│ • 存储到 Redis                │\n└──────────┬─────────────────────┘\n           │\n      ┌────┴────┐\n      │ 生成成功  │ 生成失败 ─┐\n      │          │          │\n      ▼          │          ▼\n   SUCCESS       │      ERROR\n      │          │          │\n      │          └──────────┤\n      ▼                     ▼\n┌──────────────────────────────┐\n│ 记录日志                      │\n│ logger.LogIf(err)            │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 返回 JSON 响应                │\n│ {                            │\n│   \"captcha_id\": \"xxx\",       │\n│   \"captcha_image\": \"base64\"  │\n│ } 实现代码\n```go\nfunc (c *VerifyCodeController) ShowCaptcha(ctx *gin.Context) {\n    id, b64s, err := captcha.NewCaptcha().GenerateCaptcha()\n    \n    logger.LogIf(err)\n    \n    response.JSON(ctx, (captcha.expire_time)\n  - 本地环境: 按配置 (captcha.debug_expire_time)\n    ↓\n保存到 Redis\n    ↓\n验证码前缀: \"app.name:captcha:\"\n```\n\n---\n\n### 3.2 通过手机号发送验证码\n\n#### 路由信息\n- **路径**: `POST /v1/auth/verify-codes/phone`\n- **处理器**: `VerifyCodeController.SendUsingPhone`\n- **请求体**: `VerifyCodePhoneRequest`\n- **验证**: `VerifyCodePhone`\n- **依赖**: SMS 服务、Redis、Aliyun API\n\n#### 流程图\n\n```\n┌──────────────────────────────┐\n│ POST /v1/auth/verify-codes/   │\n│ phone                         │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ VerifyCodeController         │\n│ .SendUsingPhone()            │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 创建请求对象                  │\n│ VerifyCodePhoneRequest        │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 执行请求验证                  │\n│ VerifyCodePhone               │\n└──────────┬─────────────────────┘\n           │\n      ┌────┴────┐\n      │ 验证失败  │ (中止请求)\n      ├─────────┘\n      │\n      ▼\n┌──────────────────────────────┐\n│ 验证码包处理                  │\n│ verifycode.NewVerifyCode()   │\n└──────────┬─────────────────────┘\n           │\n           ▼\n┌──────────────────────────────┐\n│ 发送 SMS                      │\n│ .SendSMS(phone)              │\n│ • 生成随机验证码              │\n│ • 通过 Aliyun API 发送         │\n│ • 存储到 Redis                │\n└──────────┬─────────────────────┘\n           │\n      ┌────┴────────┐\n      │ 发送失败     │ 发送成功\n      ▼              ▼\n┌──────────┐   ┌──────────────────┐\n│Abort500  │   │ response.Success()│\n│(错误响应) │   │ (成功响应)        │\n└──────────┘   └──────────────────┘\n      │                │\n      └────────┬───────┘\n               ▼\n        ┌──────────────┐\n        │ HTTP Response│\n        └──────────────┘\n```\n\n#### 实现代码\n```go\nfunc (c *VerifyCodeController) SendUsingPhone(ctx *gin.Context) {\n    req := requests.VerifyCodePhoneRequest{}\n    if ok := requests.Validate(ctx, &req, requests.VerifyCodePhone); !ok {\n        return\n    }\n    \n    if ok := verifycode.NewVerifyCode().SendSMS(req.Phone); !ok {\n        response.Abort500(ctx, \"Send sms fail.\")\n    } else {\n        response.Success(ctx)\n    }\n}\n```\n\n#### 验证码生成和发送流程\n\n**详细的验证码发送流程**:\n```\n1. 生成验证码\n   ├─ 生产环境: 调用 helpers.RandomNumber(code_length)\n   └─ 本地开发: 使用配置的 debug_code\n\n2. 检查调试模式\n   ├─ 非生产环境 + 手机号前缀匹配\n   │  └─ 直接返回 true (用于开发/测试)\n   └─ 其他情况继续\n\n3. 通过 SMS 驱动发送\n   ├─ 驱动: Aliyun (阿里云)\n   ├─ 参数:\n   │  ├─ Phone: 手机号\n   │  ├─ Template: SMS 模板\n   │  ├─ Data: 模板参数\n   │  │  ├─ code: 验证码\n   │  │  └─ min: 有效期(分钟)\n   │  └─ SignName: 短信签名\n   └─ 结果处理\n      ├─ 成功: 返回 true\n      └─ 失败: 记录日志, 返回 false\n\n4. 存储验证码\n   ├─ 存储引擎: Redis\n   ├─ Key 格式: \"app.name:verifycode:phone\"\n   ├─ 过期时间:\n   │  ├─ 生产环境: verifycode.expire_time (分钟)\n   │  └─ 本地开发: verifycode.debug_expire_time (分钟)\n   └─ 值: 生成的验证码数字\n```\n\n---\n\n## 4. 路由组织结构\n\n### 路由树\n\n```\n/\n└── v1\n    ├── GET  /              (健康检查)\n    └── auth\n        ├──
